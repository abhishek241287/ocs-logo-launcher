2. APP LAUNCHING
==============================

The current implementation incorrectly says "App not installed" even though Instagram, YouTube, WhatsApp, Chrome etc. are installed.

Do NOT use Linking.canOpenURL() to determine whether an app is installed.

For Android use explicit package names and Android Intents.

Use these package names:

Instagram:
com.instagram.android

YouTube:
com.google.android.youtube

WhatsApp:
com.whatsapp

Chrome:
com.android.chrome

Gmail:
com.google.android.gm

Google Maps:
com.google.android.apps.maps

Camera:
android.media.action.IMAGE_CAPTURE

Settings:
android.settings.SETTINGS

When the user taps an icon:

• Try to launch the application.
• If launching fails, print the COMPLETE Android error in the console.
• Only show "App not installed" if the package truly cannot be opened.

Do NOT hide exceptions.
